🧩 Introduction
Cuahangsach-online is an online bookstore website project developed to build a basic e-commerce system for bookstores.
The project helps users easily search, view detailed information, and order their favorite books, while also providing book data management tools for administrators.

🚀 Technology used
ASP.NET MVC
ADO.NET
SQL Server
HTML, CSS, JavaScript, Bootstrap

✨Main functions
  User
   Display a list of featured and new books.
   Categorize books by category.
   View detailed information about a book (author, price, summary).
   Add products to the shopping cart, update the quantity, use vouchers, review products.
   Place orders, cancel orders and make payments.  
  Admin
   Book inventory management (Add, edit, delete, and update book information).
   Account management (View detailed information, edit, and delete accounts).
   Monthly sales statistics.
   Order cancellation management.
   Track and manage orders.

🔽 How to Download & Install
 1. Clone the repository git clone https://github.com/Truonthanh/Cuahangsach-online.git
 2. Open the project Navigate to: cd Cuahangsach-online/Nhom06_QuanLyBanSah Open the .sln file with Visual Studio.
 3. Fix the connection string
 4. Run the project Press F5 in Visual Studio.
